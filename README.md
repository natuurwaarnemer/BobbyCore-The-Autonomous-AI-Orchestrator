# BobbyCore-The-Autonomous-AI-Orchestrator
Een open‑source AI‑ecosysteem bestaande uit drie gescheiden componenten:  BobbyCore — het autonome AI‑brein  BobbyBuddy —  het wake‑word apparaat  Home Assistant — externe smart‑home backend  Samen vormen ze Hello Bobby, een lokaal, privacy‑vriendelijk alternatief voor Alexa/Google Home — maar volledig modulair, uitbreidbaar en jouw eigendom.
